# 撮影指示書 — 付き添いサポート・介護タクシー公式サイト用

## 撮影全体の方針

- **トーン**：自然光のやわらかい逆光、美しい景色×人物の引き構図
- **表情**：笑顔または穏やかな表情。作り物感のないリアルさを重視
- **スタッフ服装**：明るい色のスクラブまたはポロシャツ＋名札。男性・女性の両方を必ず起用
- **利用者**：高齢者・車いすユーザー・医療的ケア児など多様な状況を網羅
- **NG**：いかにもストックフォトな作り物感、暗い表情、乱雑な背景

- 利用者には事前に撮影同意書を取得してください
- モデルを起用する場合は、実際の利用シーンに近い自然な演技を心がけてください

- 全カットで「清潔感」「安心感」「楽しさ」が伝わることを最優先してください
